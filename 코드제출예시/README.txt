##################################
###전처리/모델링/predict 예시 코드 설명###
##################################

1. 각 파일 별(전처리/모델링/predict) 로 들어가야 할 내용 및 형식을 담은 파일들입니다.
  1) 파일 별 맨 첫 파트 (디렉토리, 라이브러리 및 데이터 불러오기 코드)는 그대로 실행할 수 있도록 만들어 놓았습니다. 
      디렉토리 및 사용 라이브러리, 데이터 이름 등만 바꿔서 사용하시면 됩니다 !
   2) 최대한 비슷하게 형식을 맞춰주시되, 분석 과정 상 인덱스 내용이나 순서 변경이 필요 하다면 변경해서 사용해주시면 됩니다. 

2. 전처리, 모델링 등은 예시를 보여드리기 위한 코드들입니다. 코드 보다는 파일 형식에 집중해서 보시면 됩니다.
   (그대로 했다간 좋은 점수를 받기 어려울 것이에오..정말 대충한 코드니까요..예시는 예시일 뿐 !
    코드는 반드시 평가 기준을 참고해주세요!!!)

3. 전처리 파일 부터 predict 파일까지 오류 없이 실행되어야 재현성 점수를 받을 수 있습니다.
  라이브러리 실행, 파일 저장, 불러오기 등도 학회장팀 노트북에서 오류없이 실행되어야 한다는 점을다시 강조드립니다!!!

4. 사용언어는 r, python 중 하나를 선택해서 사용하시면 됩니다. python 사용 시에도 예시 파일의 인덱싱 및 주석 형식을 맞춰주세요!
